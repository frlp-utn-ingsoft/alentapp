# Changelog

Todos los cambios notables en el proyecto **Alentapp** serán documentados en este archivo.
El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/).

## [0.1.0] - 2026-05-03

### Añadido
- **Documentación Técnica (TDDs):** Se incorporaron los Documentos de Diseño Técnico para el ABM (Alta, Baja y Modificación) de las principales entidades del sistema.
  - **Member:** Registro, actualización y eliminación de socios (TDD-0001, TDD-0002, TDD-0003).
  - **Locker:** Gestión y estado de casilleros (TDD-0004, TDD-0005, TDD-0006).
  - **Equipment Loan:** Registro y control de préstamos de equipamiento (TDD-0007, TDD-0008, TDD-0009).
  - **Sport:** Administración de deportes y cupos (TDD-0010, TDD-0011, TDD-0012).
  - **Payment:** Gestión de estados de pagos (TDD-0013, TDD-0014, TDD-0015).

### Mejorado
- **Flujo de trabajo:** Se adoptó la convención *Conventional Commits* y el flujo *Feature Branch* para la integración de la documentación mediante Pull Requests.
